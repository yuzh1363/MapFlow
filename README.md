# MapFlow - 學習地圖與職涯探索平台

MapFlow 是一個旨在協助使用者規劃學習路徑與探索職涯發展的 Web 應用程式。透過直覺的視覺化介面，幫助使用者掌握技能成長進度，並探索適合的職涯目標。

## 🛠 技術棧
* **框架**：Vue 3 (Composition API)
* **建構工具**：Vite, Vue Router
* **狀態管理**：Pinia
* **樣式**：Bootstrap 5, SCSS
* **後端/資料庫**：Firebase

## 🚀 我的核心貢獻
* **動態資料管理**：利用 **Pinia** 整合 **Firebase** 進行非同步資料存取，實作首頁搜尋卡片的動態渲染。
* **搜尋功能邏輯**：與 AI 協作開發搜尋功能的過濾邏輯，提升使用者查找學習資源的效率。
* **頁面與組件開發**：獨立完成搜尋首頁主要卡片內容區塊、角色樹洞、樣板市集等頁面，並封裝多項高複用性的 UI 組件（如卡片、按鈕組件）。
* **結構規劃**：實作 **Nested Routing (巢狀路由)**，建立層次分明的頁面結構，優化使用者導覽體驗。
